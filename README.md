Программа выдает список всех продуктов.
Данные берутся посредством Hibernate из БД.
Есть также возможность добавлять новые продукты, редактировать существующие и удалять их.


***Технологии:***<br>
Java <br>
Spring Boot <br>
Hibernate <br>
MySQL <br>
Thymeleaf<br>
Html и Css<br>
JUnit <br>
Git<br>
